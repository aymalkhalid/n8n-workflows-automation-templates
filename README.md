# ⚙️ n8n Workflow Automation Templates

Welcome to my repository of **n8n workflow templates**! This collection contains ready-to-use JSON workflows designed to automate tasks, connect APIs, and streamline operations. 

Whether you are building basic automations or complex multi-step processes, you can use these templates as a starting point.

## 📂 Repository Structure

## 🚀 How to Use These Workflows

Importing these workflows into your own n8n instance is quick and simple. Choose one of the methods below:

### Method 1: Copy & Paste (Fastest)
1. Open the folder of the category you want (e.g., `Whatsapp Integration`).
2. Click on the specific `.json` workflow file.
3. Click the **Copy raw contents** button (the two overlapping squares icon at the top right of the code box).
4. Go to your n8n workflow canvas and simply paste (`Ctrl+V` or `Cmd+V`). The nodes will automatically generate!

### Method 2: Import from File
1. Download the `.json` file from this repository to your computer.
2. Open your n8n workspace.
3. Click the workflow menu in the top-left corner.
4. Select **Import from File...** and upload your downloaded `.json` file.

## 🔒 Security & Credentials
All workflows shared in this repository have been scrubbed of personal API keys, webhooks, and passwords. 

When you import a template, you will need to configure the credentials on the respective nodes using your own accounts to make the automation work.
